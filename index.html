Variables

Variable names must start with _, $, or any letter, and cannot be a number. Usually preceding a name with _ denotes it as a private variable, and $ is for auto-generated code.

Variables can be declared using var, let, or const. Generally var should not be used, mostly to make sure we catch errors for uninitialised variables.

Example Code

          const applicationId = "Some GUID";
          let counter = 100,
              otherVariable = 200;
          
          counter = 99;   // Changes counter to 99
          counter -= 5;   // Subtracts 5 from counter
          counter++;      // Incredements counter
